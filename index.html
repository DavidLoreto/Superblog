Practicas del curso de Git y GitHub

Comandos utiles:

git init: con este comando podemos inicializar un repositorio.

git add: con este comando indicamos a que archivos le a hacer tracking de sus cambios.

git commit: este comando sirve para guardar los cambios en el registro hisotrico del repositorio.

git show: al usar git show sin especificar un archivo puedo ver los cambios hechos al repositorio en el ultimo commit.

git remote: con git remote podemos gestionar los repositorios externos

git pull: este comando se usa para traer datos de un repositorio externo

gut push: con este comando enviamos datos a un repositorio externo

git diff: al usar este comando podemos ver de forma sencilla las direfencias entre 2 commits, archivos, etc.

Consejos

al crear un repositorio en GitHub y intentar hacer un git pull a hacia repositorio ya existente se general el error "unrelated Histories" para solucionarlo se usa el comando:
git pull --allow-unrelated-histories origin branch
